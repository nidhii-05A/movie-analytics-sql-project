# Movie Analytics Database Project

## Overview
Developed a relational movie analytics database using MySQL to manage movie, actor, and review data.

## Technologies Used
- MySQL

## Database Tables
1. actors
2. movies
3. reviews

## SQL Concepts Used
- CREATE TABLE
- INSERT INTO
- SELECT Queries
- JOIN Operations
- GROUP BY
- Aggregate Functions
- ORDER BY
- Filtering Queries

## Analytical Queries
- Highest rated movie
- Average movie ratings
- Review analysis
- Actor-movie relationship analysis
- Movie release trends

## Project Outcome
Built a structured relational database and performed analytical SQL queries to generate meaningful movie insights.
